# Markdown Editor

Android と iOS で動作する、ユーザーフレンドリーなマークダウンエディターアプリです。

## 特徴

- 📝 **ツールバー**: 見出し、太字、斜体、罫線、箇条書き、コードブロック、引用、リンクをワンタップで挿入
- 👁️ **プレビュー**: マークダウンをリアルタイムで整形表示
- 🌙 **ダークテーマ**: 目に優しいプレミアムなデザイン

## 技術スタック

- React Native + Expo
- TypeScript
- react-native-markdown-display

## インストール

```bash
npm install
```

## 起動

```bash
npx expo start
```

Expo Go アプリでQRコードをスキャンして実機確認できます。
